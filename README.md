# xiaokeer-project-arrange

## 项目介绍

xiaokeer项目生态中,整理其它项目的项目,可以让1个混乱的项目变成xiaokeer开发规范的项目，最终达到github可开源的程度
截至目前xiaokeer电脑中大大小小的有几十个项目，它们都不符合xiaokeer项目的开发范式，无法开源，无法共享，无法形成生态。
因此，此项目应运而生
